Student Resources for HTML

  1. www.learn-html.org

    This one starts with the basic Hello World program like with any programming language. The great aspect of this one is the ability to test some code, right there on the page, just like w3schools. It starts with the basics but works up to things like bootstrap, etc.

  2. html.com

    I like this one because it has different levels for where you can begin, like beginner and advanced. It helps you create your webpage in minutes, not hours, days, or months.

  3. www.codecademy.com

    This is a good one for HTML and other languages and is free, which is great. You do have to create a username and password. It does have the ability to upgrade to Pro

  4. learn.shayhowe.com/html-css/

    This is a free one that helps you build your first webpage and then adds components as you go, a lot like HTML CSS class. The format and fact that the information is right up front make this a resource I am likely to use again.

  5. teamtreehouse.com/learn/html

    While this one is not free, it has a free trial so you can give it a try. There is a review that states how the videos help with the "Aha" moments. This website provides a great foundation. You can build your own projects in about 6-12 months.
